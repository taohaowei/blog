1. const作用是什么？
const关键字（翻译为：常数，不变的;）被它定义的变量为常量（不可再赋值的变量为常量）
下面我们来看一段简单的代码，

int main(void){
    int const a = 10;//定义一个常量a=10;
    //上述等于  const int a = 10;下文中用第一种表示方法，比较直观。
    a++;//编译时报错，因为const形容了a这个变量，它不可再变化
}

运行结果报错：[Error] increment of read-only variable ‘a’
可以看到，被const形容过的变量a为只读（read-only）。

2.const形容指针两种表示形式。

在描述指针的形式前，先来引领一下思路，便于后面理解。
`int a = 10;` 表示a变量被int修饰,也就是整形变量a
按照上面的思路，下面这段代码

int const a = 10;
```是不是就代表着，a变量被const修饰成常量(read-only)，同时！也被int修饰，所以a变量也就是整形常量
